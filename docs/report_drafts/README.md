# Report Drafts

This folder holds the working report copies that mirror the finalized submission reports.

Current draft assets:

- `internship_report_draft.md`
- `journal_paper_draft.md`
- `report_completion_notes.md`
- `figures_for_report/README.md`
- `tables_for_report/report_tables.md`

How to use these drafts:

- Start from `internship_report_draft.md` for the main internship report narrative.
- Use `journal_paper_draft.md` for the shorter journal-style version.
- Use `tables_for_report/report_tables.md` for copyable dataset and results tables.
- Use `figures_for_report/README.md` to choose the strongest existing figures.
- Use `report_completion_notes.md` for the completed export status and canonical final-report locations in `docs/final_reports/` and `AdvancePractice/reports/`.
