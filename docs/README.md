# Docs Guide

The `docs` folder is organized by document purpose.

## Folders

- `project_brief/`
  Official project brief, final report template, proposed method, and archived plan note.
- `literature_review/`
  Literature review source documents and summaries.
- `analysis_notes/`
  Working notes, interpretation notes, and project understanding notes.
- `report_drafts/`
  Reserved for final report drafts and journal-paper drafts.

## Current Best Entry Points

- Brief: `docs/project_brief/Project Description UMN X Teeside 2026 - Nabila.pdf`
- Report template: `docs/project_brief/Final Report Template.docx`
- Method draft: `docs/project_brief/Proposed_Method.txt`
- Literature summary: `docs/literature_review/Summary_Literature_Review.docx`
- EDA note explanations: `docs/analysis_notes/graph_explanations.txt`
