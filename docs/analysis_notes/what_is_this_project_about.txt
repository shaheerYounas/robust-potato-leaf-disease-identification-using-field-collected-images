What students usually do for a project like this:

Start with a focused literature review on plant disease detection and real-world agricultural image datasets.
Explore the provided dataset carefully: class balance, image quality, lighting variation, background noise, and label structure.
Build 2 to 4 baseline models, usually CNNs like ResNet/EfficientNet/MobileNet, and sometimes one Vision Transformer for comparison.
Train, validate, and compare them using metrics like accuracy, precision, recall, F1-score, and confusion matrix.
Add robustness improvements such as augmentation, class balancing, transfer learning, regularization, and image preprocessing.
Pick one best model and deploy it as a simple demo, usually with Streamlit, Flask, FastAPI, or ONNX.
Write the final work as a journal-style research report, not just a software project report.
